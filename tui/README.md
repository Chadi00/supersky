# Supersky TUI

Minimal fullscreen OpenTUI prototype for the `supersky` project.

## Commands

```bash
bun install
bun run dev
bun run test
bun run typecheck
```

## Interaction

```text
Arrow keys / j / k / tab  Move between sections
1-4                       Jump to a section
q or Esc                  Exit cleanly
```

Built with the OpenTUI React reconciler and scaffolded via `create-tui`.
